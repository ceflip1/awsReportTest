# awsReportTest
API that allows to store data from JSON and download those data as reports.

        Requirements
python (Python 3.6 up to 3.9 supported)
pip (Python package manager)
Docker

        Setup
-npm install
-npm install -g serverless


        Deploy
-docker-compose up
-serverless deploy --stage local

        serverless deploy
